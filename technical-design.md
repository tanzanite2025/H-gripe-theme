# Technical 页面结构草稿（/guides/technical）

## 1. 页面定位

- **路由**：`/guides/technical`
- **标题**：`Technical`
- **目标用户**：已经有一定经验的车友 / 需要精确参数的客户
- **主要用途**：
  - 集中展示与车圈、辐条、花鼓、编法、张力和计算公式相关的技术信息
  - 帮助用户在下单前快速确认关键技术参数

---

## 2. 页面整体布局（单页 + 内部分块）

- 不增加新路由，所有内容都在同一个页面 `/guides/technical` 中完成
- 页面大致结构：
  - 顶部：`Technical` 标题 + 简短介绍
  - 紧跟一行 **页面内导航 Tabs**（不跳转路由）
  - Tabs 下方是对应的内容模块（可以是：
    - 整页长文 + Tabs 只是锚点导航，或
    - 只显示当前 Tab，其他模块隐藏）

## 4. 各模块内容大纲（可自由增删）

> 下面是建议内容，你可以在这里直接改标题、顺序，或删掉不需要的点。

### 4.1 Rims（车圈） `#rims`

- **Section 标题示例**：`About Rims` / `Rims (车圈)`
- **建议子内容：**
  - 车圈直径标准（ISO/ETRTO 标注，如 622, 584 等）
  - 内宽 / 外宽，对轮胎兼容性的影响
  - 孔数（28H / 32H / 36H …）与用途建议
  - ERD（Effective Rim Diameter）的定义与测量方法
  - 与 Spoke Calculator 的关系：ERD 应如何填写

---

### 4.2 Spokes（辐条） `#spokes`

- **Section 标题示例**：`Spokes` / `Spokes (辐条)`
- **模块目标**：介绍辐条的基础知识，帮助用户理解不同辐条规格、材料和品牌在实际使用中的差异。
- **建议子内容：**
  - 常见辐条规格（直径、形状：直拉 / 弯头、圆辐 / 扁辐）。
  - 常见辐条类型与品牌（例如竞赛型、耐久型等），以及适用场景（公路 / 山地 / 重载等）。
  - 辐条长度的影响：长度过长 / 过短可能带来的风险与装配问题（例如螺纹露出太多、拧不到位等）。
  - 和 Tanzanite 自家轮组搭配时的一些推荐选择思路（可选）。

---

### 4.3 Spoke length（辐条长度计算参数） `#spoke-length`

- **Section 标题示例**：`Spoke length` / `Spoke length (辐条长度)`
- **模块目标**：系统解释计算辐条长度时会用到的每一个参数，让用户在使用 Spoke Calculator 前，知道这些数字从哪里来、代表什么含义。

详细解释辐条长度计算中用到的每一个参数，建议逐条说明：
- **Rim ERD（Effective Rim Diameter，有效圈径）**：
  - 定义：辐条头座落在圈内的有效直径，用于计算辐条长度的基础尺寸。
  - 建议：说明如何测量或从 Tanzanite 圈规格中查到 ERD，与外胎适配关系主要在 Rims 模块中展开。

- **Hub flange diameter（花鼓法兰直径）**：
  - 定义：花鼓法兰上辐孔中心构成的圆的直径。
  - 建议：描述常见范围，以及在厂商图纸中对应的标注名称。

- **Left / right flange distance（左右法兰到中线距离）**：
  - 定义：从花鼓中线到左 / 右法兰中心的距离。
  - 建议：说明该距离如何影响张力比和轮组偏移（dish）。

- **Spoke hole count（辐孔数）**：
  - 定义：圈和花鼓上的孔数（如 28H / 32H / 36H）。
  - 建议：强调圈与花鼓孔数必须一致，对应不同用途的推荐孔数。

- **Lacing pattern / cross count（编法 / 交叉数）**：
  - 定义：辐条跨过几根辐条（1x / 2x / 3x / 4x），以及径向编法的特殊情况。
  - 建议：简要说明交叉数对辐条长度计算公式的影响，必要时可以配合图示或在本节中做扩展说明。

- **Dish / offset（轮组偏移 / 偏心）**：
  - 定义：轮圈中心相对花鼓中心的偏移量（尤其是后轮驱动侧）。
  - 建议：简要说明偏移如何进入计算，具体公式可放在 Tension 模块或专门的公式小节。

- **与 Spoke Calculator 的连接**：
  - 列出用户在使用 Tanzanite Spoke Calculator 时需要事先准备的这些参数，并标明它们在计算器表单中的字段名称。

---

### 4.4 Hubs（花鼓） `#hubs`

- **Section 标题示例**：`Hubs` / `Hubs (花鼓)`
- **建议子内容：**
  - 花鼓法兰到中线距离（left/right flange distance）
  - 法兰直径（flange diameter）
  - 孔数与车圈匹配（必须一致）
  - 轴规格 / 飞轮标准（只做简要提示，不扩展为产品介绍页）
  - 如何从花鼓厂商提供的规格图中读出这些参数

---

### 4.5 Tension & formulas（张力与公式） `#tension`

- **Section 标题示例**：`Tension & formulas` / `Tension & formulas (张力与公式)`
- **建议子内容：**
  - 不同类型轮组的推荐张力范围：
    - 前轮 / 后轮
    - 左侧 / 右侧（驱动侧 / 非驱动侧）
  - 张力计读数与实际张力的换算：
    - 可以用一张简单表格列出（型号 / 读数 / 对应张力）
  - 轮径、花鼓偏移、张力比之间关系的文字说明（必要时可以有一两张示意图的说明文字）
